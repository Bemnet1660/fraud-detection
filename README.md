# Fraud Detection for E‑commerce and Bank Transactions – Task 1

## Project Overview

This repository contains the Task 1 submission for the Week 5&6 Challenge at 10 Academy.  
The goal is to preprocess, enrich, and prepare two highly imbalanced fraud detection datasets:

1. E‑commerce transactions (Fraud_Data.csv) – enriched with geolocation and behavioural features.
2. Bank credit card transactions (creditcard.csv) – PCA‑transformed features.

Task 1 focuses on:
- Data cleaning and exploratory data analysis (EDA)
- IP‑to‑country geolocation enrichment
- Feature engineering (temporal, velocity, frequency)
- Data transformation (scaling, encoding)
- Handling class imbalance only on the training set using SMOTE

The output includes cleaned datasets, train/test splits with resampled training data, and a detailed report.
